# Homebrew Tap for RainDrop

macOS 집중 타이머 앱 RainDrop의 Homebrew Cask 저장소입니다.

빗물을 모아 양동이를 채우는 컨셉의 포모도로 타이머. 6종 양동이 스킨, 무한 모드, 백색소음, 집중 확인 알림 등을 지원합니다.

## 설치

```bash
brew install --cask mgm136044/tap/raindrop
```

## 업데이트

```bash
brew update && brew upgrade --cask raindrop
```

## 삭제

```bash
brew uninstall --cask raindrop
```

## 주요 기능

- 집중 타이머 + 양동이 채움 시스템
- 무한 모드 (끝없는 집중, 순환마다 코인 적립)
- 6종 양동이 스킨 (나무 → 무지개)
- 백색소음 (rainymood.com 연동)
- 집중 확인 알림 (10초 타임아웃)
- 스티커 꾸미기 상점
- 월별 히스토리 달력
- 메뉴바 타이머
- 자동 업데이트 확인
